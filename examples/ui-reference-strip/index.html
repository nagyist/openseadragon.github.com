Image Reference Strip

An image reference strip allows the user to quickly preview and navigate a sequence of images. This feature works only with sequence mode enabled.

Horizontal Scrolling Image Reference Strip

Horizontal Scrolling Image Reference Strip

The relevant configuration options are shown below. 

OpenSeadragon({
    ...
    sequenceMode: true,
    showReferenceStrip: true,
    ...
});

Vertical Scrolling Image Reference Strip

Vertical Scrolling Image Reference Strip

The relevant configuration options are shown below. 

OpenSeadragon({
    ...
    sequenceMode: true,    
    showReferenceStrip: true,
    referenceStripScroll: 'vertical',
    ...
});
